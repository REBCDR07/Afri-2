My AfriChat

Widget chatbot IA conversationnel avec synthèse vocale — le bot répond en texte et en audio. Installable sur n'importe quel site en une ligne npm. Streaming SSE, WebRTC temps réel, 6 voix disponibles. Première du genre au Bénin et en Afrique francophone. Développé pendant le stage Build With Afri.

npm install my-africhat React TypeScript TTS API Supabase Edge Functions SSE / WebRTC
npm package Demo live
// 1. Install
npm install my-africhat

// 2. Configure africhat.config.ts
const config = {
  branding: { accentColor: "#f59e0b" },
  assistant: { voice: "nova", audioEnabled: true },
  api: { siteKey: "afc_live_..." }
}

// 3. Mount
import { createMyAfriChat } from "my-africhat"
createMyAfriChat(config).mount()
